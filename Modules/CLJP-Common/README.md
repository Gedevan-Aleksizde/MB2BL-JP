# 概要

1.  バニラで日本語設定にするとあちこち文字化けするので文字化けしないフォントに置き換えます. いくつか種類を用意しました.
2.  日本語のテキストも変な箇所が多いので修正します. 約2万件ある本体のテキストのうち, **約27%**が修正されます.
    -   当初は単純な誤字誤訳の修正だけを想定していましたが, 修正箇所の多さや原文自体の曖昧さと誤り, 修正後の整合性をも考慮するとさらなる修正が必要になり, 結果としてかなりの部分の印象が変わります.
    -   現時点では一般セリフはまだほとんど確認してませんが, メニュー画面やシステムメッセージでよく見られる文やアイテム名等を重点的に変えたので印象はだいぶ変わると思います.
    -   これ以降も何度かに分けて更新する予定です

アーリーアクセスの頃に似たようなことやってる人々の形跡は見つけましたが今公開されてて生きてる/まとまったものが見つからなかったので作りました.

## このModを更新する場合

テキスト修正は段階的に行うので, 何度か更新が必要だと思います. 一旦このmodをインストールした場合, 以降の更新は以下の手順で行うことをおすすめします. (初回と同じ方法でも一応できます)

1.  Main Files の `CorrectLocalizationJP-Text` だけをダウンロードしてください.
2.  ダウンロードファイルを解凍し, `CorrectLocalizationJP-Text` フォルダをマウントアンドブレード2のインストールフォルダ内の `Modules` フォルダ内の同名のフォルダで上書きしてください.
3.  ランチャーでModが2つとも有効になっていることを確認してゲームを起動してください
4.  (v0.9.0 を使用していた方へ) v0.9.1 以降は言語設定で「正しい日本語」を選ぶメリットはありません. 後方互換性のためしばらく残しておきますが, 「日本語」に変更しても問題ありません.

(初回版 v0.9.0をインストールした方へ) v0.9.0 はミスでバージョン番号をv1.0.0にしていました. 更新でバージョン番号が戻っていますが問題ありません.

解説: 段階的な修正が予想されるので, テキスト変更用ファイルとフォントデータを分割して配布することにしました. フォントデータはかなり大きい割に更新することがほぼないと思うので, テキストファイルのみでダウンロードできればかなり時間短縮になると思います. いちおう本体にもテキストファイルは入れるようにしますが, 更新し忘れるかもしれません. 表示されているバージョン番号に注意してください.

## このmodを使用しつつ, 他の日本語化modを使用したい場合

1.  このModを有効にしたまま, 言語オプションで「正しい日本語」ではなく, デフォルトの「日本語」に戻してください.

-   ゲーム本体のファイルを上書きして日本語を変更するタイプのmodをインストールする際は, CorrectLocalizationJP-Text 以下にバニラのモジュールと同じ名前のフォルダがあります. それにXMLファイルを上書きしてください. なければ通常のモジュールフォルダのファイルに上書きしてください.

# Mod の詳細

## フォントの修正について

M&B2 本体は現在 (v1.0.3), ところにより漢字が文字化けして四角形になったり (いわゆる「豆腐化」), スタイルの全く違う KaiTi フォントにフォールバックしたりして, かなり読みづらいです. そのため使用フォントを適切なものに置き換えます.

フォントはいくつかバリエーションを用意しました. 好きなものをどれか1つインストールしてください.

1.  「源真ゴシックP」 - バニラの日本語フォントに近い形状
2.  「源暎ラテミンP」 - 雰囲気重視

</details>

## 翻訳テキストの修正 (あるいは校正) について

v1.0.3現在, 誤訳だけでなく, 大意は掴めますが, 全体的に不自然でわかりにくい日本語が多いです. このmodのタイトルは Correct Localization ですが, 狭義の誤訳より広い範囲を修正対象としています. 現時点で, 修正が必要と判断した箇所は全体の **27%**程度になります. おおまかな修正方針を挙げると, 以下のようになります.

-   明確な誤訳 (そこそこあります) を修正
-   誤字脱字 (わりとあります) を修正
-   そもそも元の英文からして間違っている・ミスリーディングなので訳文もおかしくなっている (けっこうあります) を, なるべく正確で明瞭な文に修正
    -   よって, 純粋な「翻訳」の範囲を超えて「テキスト限定のバグフィックス」的な要素もあります
-   統一されていない訳語 (多いです)
    -   一見すると日本語として問題なさそうなものでも, 訳語が統一されていないものが多くあります
-   システムメッセージやアイテム名に見られる悪文 (とても多いです) を修正し, 規則性をもたせ読みやすい語順にする
    -   これの多くは上記「原文からして推敲されていない」という問題におそらく関係しています
    -   特にアイテム名は意味が全く変わっているものが多いので, わかりやすさ, シンプルさ, そして正確さを重視した名称にします
    -   パークスキル説明文やステータス表示で「ゴールド」「デナール」「XX減少」「XX低減」「XX軽減」といった統一されていない文言を読みやすくします
-   文体の統一
    -   ですます調とである調が交互に現れたり, 急にくだけた口調になったりする説明文を修正
    -   システムメッセージの表現を統一し, 過去形だったり他人事だったり宣伝口調だったりするのを修正
    -   ポップアップヘルプ等はですます調, ログは簡潔さを重視した文体にしています
-   インターフェースの整備
    -   英語的な語順になっていたり, 語順が統一されなかったりするため, テキストの修正で可能な範囲で, わかりにくい箇所を変更します
-   一部のユニット名を識別しやすく変更
-   地名・人名・固有名詞の表記の統一 (現時点では, 部のみ)
-   確認の必要な箇所が膨大なので今後も何度か修正版を更新するかもしれません.
-   より詳細な校正方針は長くなったので別の機会に書きます. 修正の典型例は比較画像を見てください

表示箇所別に分けると, 以下のような進捗状況になっています. 数字は体感です. 数字と実態とに大きくずれがある場合, それは内部IDの重複が主な原因です

1.  メニュー画面, マップや戦闘時のステータス表示・UI関連 - 95% <s>90%</s> ほぼ完了
    -   原文が曖昧, あるいは誤っている可能性が高いために適切な修正が保証できない箇所が一部あります
2.  オプションの設定項目関連 - <s>50%くらい?</s> 完了
3.  カスタムバトル - たぶん完了
4.  アイテム名・鍛冶の部品名 - 95% <s>80%</s>, おおむねできていますが, 一部確認漏れがあるかも
    -   必要に応じて実物を確認した上でわかりやすい表現に直し, ある程度規則性のある命名に整備しました.
5.  キャンペーンログ - ??
    -   単なる翻訳文の問題以外の原因で, 表示がおかしい箇所があるため調査中
6.  クエスト・会話セリフ・百科事典のテキスト - 10% 正しくない用語や誤字を機械的に置換した程度でほとんど手つかず
7.  固有名詞・汎用人名のスペル確認 - 1% ほぼ機械的な置換のみ
8.  <s>マルチプレイ - ??%, Module版では動作しない・そもそもあまりやってないため確認中</s> マルチはシングルプレイ以上にmodの挙動がよくわからないので一旦保留とします
9.  ミニゲームのルール説明 - 0% 一度もやってないのでほぼ未確認です

-   アイテム名の大幅な変更
    -   単なる誤訳, そもそも元の名称がわかりにくい, 書き間違えたりしている, といった様々な理由により, アイテム名, 特に防具と鍛冶パーツの名称の多くは修正後はかなり元の印象から変わります
    -   基本的には, 命名ルールの一貫性を特に重視して直しています
    -   名前が長くなりすぎると表示がおかしくなるおそれがあるので, 今のところは全体的にカタカナ語は少なめです. よって “lamellar (armor)” が「薄金鎧」になったりします,
    -   skullcap -&gt; スカルヘルムのような謎造語も差し替えています
    -   例: 『Heavy Lamellar over Mail/**鎖帷子付きのヘビーラメラー**』-&gt;『**重い鎖仕込みの薄金鎧**』 (語順の統一と中途半端なカタカナ語を修正)
    -   例: 『Decorated Leather Harness over Scale/**鱗下地の装飾された革の引き具**』-&gt; 『**装飾された革のハーネス付きの鱗防具**』 (これは公式版の訳では Harness を馬具と誤解しています)
    -   アイテム名の変更ルールは説明すら結構長くなってしまうので今回は詳細説明を省略します
-   各兵士クラスの名称変更
    -   簡潔さや, ツリー上の繋がり・逆に異なる勢力どうしの区別のしやすさを重視して変更しています
    -   原文の時点で紛らわしい表現がいくつかあり, 結果として訳語にも影響しているものが多いため, それらは大きく変わります
        -   小規模勢力と賊系ユニットは特に雰囲気が変わります (画像参照)
        -   例: Looter/略奪者 -&gt; 追い剥ぎ, Highwayman/追いはぎ -&gt; 街道強盗
    -   「帝国」「アセライ・」といった接頭語を, 紛らわしくならない範囲でなるべく省きます.
        -   例: 「アセライ・マムルーク重騎兵」-&gt;「重装マムルーク騎兵」, 「帝国カタフラクト」-&gt;「カタフラクト」
    -   逆に「XXの新兵」「YYの歩兵」といったどの勢力にもいそうな低レベル帯のユニットは規則性のある表記にしています
    -   翻案の色が強いですが, 作中で言及されている範囲で一部名称を大きく変更します (いわゆる, 「ロアフレンドリー」な範囲で)
        -   例: Vlandian Vanguard/ヴランディア前衛兵 -&gt; アヴァント・ギャルド (Vanguard とつく兵士は他にもいますが, 作中で元ネタを察せる設定がなされているため)
    -   見慣れない固有名詞は, 補足説明になるような名称にします (いわゆる重語表現)

# 既知の不具合

1.  文字が重なって表示される箇所がある
    -   源暎ラテミン版はメニュー画面等ではあまり気になりませんが, キャンペーンマップや戦闘中のログでは文字の行が重なって見づらいことがあります
    -   既に知られているように, .fnt ファイルの数字を書き換えて文字を小さくすれば重ならなくなりますが, 私の試した範囲では, ログの文字が重ならなくなる程度まで文字を小さくすると逆に他のレイアウトが崩れることが多いです. 何度か試してどの場面でもレイアウトが極端に崩れてなさそうな文字サイズで配布しています

    -   スタイルを修正すれば解決しそうですが, 公式ドキュメントの解説が充実してないので今のところは修正していません.

# 想定問答集 (FAQ) とトラブルシュート

-   Q.: マジで文字化けしなくなるの?

-   A.: 通常の日本語であれば多分大丈夫です. 以下, 詳細を折りたたみ
    <details><summary>クリックで展開</summary>例えば仮に, modの翻訳文に「『颯』爽」「樽」「隼」「憂『鬱』」「『驃』騎兵」といった常用漢字外の字を追加したとしても表示されます. 厳密なテストはしていませんが, 『源真ゴシック』は 19,623, 『源瑛ラテミン』は 18,198 のグリフを収録しています. いずれもAbode Japan 1-6 規格に準拠した『源ノ角ゴシック』フォントを元にしているため, 通常日本語で使われうる文字はほぼ全て対応していると考えて問題ないです. 逆に, 簡体字などは対応してないですが, AJ1-6のグリフ数でもけっこうギリギリだったので現状M&B2 の仕様に沿ってCJK統合漢字を全て同じスタイルで表示させるのは困難だと思います. このように, 日本語で使われる文字はほとんど対応しているはずなので, 他のmodのテキストを翻訳している方は常用漢字縛りを気にすることなく自由に訳してください.</details>

-   Q.: 他の日本語化modと併用できる?

-   A.: 手間がかかることがありますが, 注意すれば可能です. 以下詳細を折りたたみ.
    <details><summary>クリックで展開</summary>
    Modを上書きして日本語化するタイプのファイルの場合,「日本語」と本modの「正しい日本語」は別言語扱いなので, 「正しい日本語」選択時にはModの日本語化がされない可能性があります. 翻訳ファイルの XML ファイルの id を全て “日本語” から “correct\_正しい日本語” に修正するか (VS Code のフォルダ単位の置換機能とか使ってください), 本modの上書き版を使用すれば併用できると思います. 本体の日本語を変更するタイプのmodは, 場合によります. 私がNexusで見つけられたのは [Historical Unit Name Japanese MOD](https://www.nexusmods.com/mountandblade2bannerlord/mods/2806) mod の1つだけで, このmodであれば本modの上書き版の後にさらに上書きすればユニット名だけ変わると思います. もしくは Module 版とうまく手作業で連結してください.

    また, DL数の多いModであっても, 言語フォルダを使わずにテキストをハードコードしている人が結構多いです. この場合も本体の日本語が上書きされたり, 本体用の日本語テキストがModで追加されたテキストにでたらめに割り当てられてしまうことがあります. ケースバイケースなので, 一般的な対処法を説明するのは難しいです.
    </details>

-   Q.: ゲームが動かない, エラーが出て落ちる. どう落とし前つけてくれるんじゃ

-   A.: 長いですがこうしてください.
    <details><summary>クリックで展開</summary>
    以下は私の個人的経験に基づくものなので, 完全ではないかもしれません. この mod とは関係ない, 本体のバグが原因であるケースもあるかもしれません.

    ゲーム本体のバージョンと Mod にかかれている対応バージョンが一致しているか確認してください. バージョン番号の頭に e とついているものはアーリーアクセス版なので, このmodはおそらく動きません. また, 本体上書き版はバージョンが一致しない時の警告が出ない可能性が高いので注意してください. バージョンが違っても動作することはありますが, 動作保証はできません. こういう場合, ゲームがクラッシュする, 動作はするが日本語修正が部分的におかしい (一部英語だったり一部バニラの日本語のままだったり) など, 様々な不具合の可能性があり, 何が起こるかわかりません.

    次に, ランチャーで全てのmodを無効化してから起動してみてください. これで動くなら, いずれかの mod が原因の可能性が高いです. もしそのmodが私のmodだけなら, 具体的な状況を教えていただけると助かります.

    それでも問題が解決しない場合, 本体の再インストールが必要なときもあります. はじめは Steam の「整合性の確認」だけでも十分です. それでもだめなら再インストールしてみてください. その場合, Steamからアンインストールボタンを押した後にインストールフォルダに残っているファイルがあったらそれも削除してください. (インストールフォルダだけでいいです. セーブデータまで削除するのは「最後の手段」にしてください)

    実は, M&B2 は公式にアナウンスのあるアップデート以外にも, 致命的なバグ修正のために通知なしに更新することがあります (通称, サイレント/ステルスアップデート). この変更によって, 同一バージョンでも中のプログラムが違うためmodが動かないということが稀にあるようです. このmodは現状テキストファイル書き換えとフォントの置き換えだけなので, 影響はあまりないと思いますが, もしバージョン番号が適切なのに動作がおかしいときは, 本体を再インストールするといいかもしれません.
    </details>

-   Q.: 文字化けは回避したいけどお前の翻訳センスないから文章まで変えてほしくない

-   A.: このModフォルダ内にある `ModuleData` フォルダを削除すると, テキストは変更されず, フォントだけが変更されます

-   Q.: 翻訳テキストいらんとこまで変えないで

-   A.: たとえ問題のある訳であるからと, 固有名詞やスキル名も変えるとかえって混乱を招く可能性があるのは理解しています. 選択制にするかもしれませんが, 作業量が多いので期待はしないでください
    <details><summary>自分で変更したい人向けのヒント</summary>
    仕様上, 場面ごとにカスタマイズするのは難しいですが, 例えば名前が `std_spitems` で始まるXMLファイルにアイテム名が集中しています. ユニット名ならば `std_spnpccharacters` で始まるファイルです. `std_mp` と `std_multiplayer` で始まるものはマルチプレイ用のテキストです (おそらく皆さんが気にするのはだいたいここでしょう). これらを差し戻せば, まとめて公式テキストに戻ります. ただし, IDの重複がかなり多いので, 1つ変更しても反映されないことが結構多いです. 全体の2, 3割はあります.
    `</details>`

-   Q.: 「退役勧告」って誤訳じゃないの?

-   A.: 剣に「アーリー・リタイアメント」なんて名前をつけた人に言ってください. 作中でもかなり強い部類の剣なので, これでもかっこよくしたつもりです。 IDは “x0x1gIbT” なので気に入らなかったら適当に改名してあげてください.

-   Q.: フォントもいらないんだけど

-   A.: このModフォルダ内にある `GUI`, `AssetSources`, `Assets`, `RuntimeDataCache` フォルダを削除すればフォント変更はされなくなります. ただし, 私の作成した文章はデフォルトの日本語フォントが対応していない文字を使っているので文字化けすると思います.

-   Q.: `RuntimeDataCache` っていうやたらとサイズの大きいフォルダがあるんだけどCacheって名前だしこれいらなくね?

-   A.: これがないとフォントが正常に読み込まれません. なぜこのような仕様なのか私もよくわかりません.

# 補足/Notes

-   NexusMods の掲示板には日本語が書き込めません. 日本語での不具合報告は [GitHub](https://github.com/Gedevan-Aleksizde/MB2BL-JP) か[私のブログ](https://under-identified.hatenablog.com/entry/2023/01/24/232416)でお願いします. ただし, クラッシュや強制終了時のエラーコードの原因特定までは対処できない可能性が高いです (まだまだ本体由来のバグが多いです). 主にテキストが間違ってる, 正常に表示されない箇所があるといった問題の対応になります.
    -   英文では同じ文でも日本語では文脈に応じて訳が変わる箇所がありますが, たまに同じ文を使いまわしている箇所があるために片方の表示箇所に合わせて訳した結果, もう一方の訳が修正前よりも逆におかしくなることがたまにあります. 全て自力で発見するのは難しいので, 見つけましたら一報いただけるとありがたいです
    -   基本的に実際にプレイするしかテスト手段がないので, 細かいミスは多いと思います
-   フォントサイズはわずかに小さく調整しましたが, もしかすると文字がUIの枠からはみ出て操作しづらい箇所が発生するかもしれません. これも発見されましたら報告していただけると助かります.
-   私 (このmodの作者) はM&B2をEAの比較的早い段階で購入し, その時点で日本語対応が計画されているとアナウンスされていたので日本語化コミュニティには参加していませんでした. 日本語化された安定版が出たと聞いて久々に再開したらあんまり出来がよくなかったのでこれを作りました. そちらのものは確認してないので翻訳の方針が違うかもしれません.
    -   別ゲームの話になってしまいますが, 当初はスカイリムの [Improved Japanese Translation](https://tktk1.net/skyrim/mymod/improved-japanese-translation/) のような公式日本語訳の修正・校正を想定していましたが, システムメッセージから先に手を加えたため, 結果として一部に見られる公式英語・日本語両方の曖昧さの解消が必要となったので, むしろテキスト限定の USKP の様相を呈しつつあります.
-   既に TaleWorlds のフォーラムで[日本語ローカライズの問題について開発元に指摘してくださっている方がいます](https://forums.taleworlds.com/index.php?threads/some-japanese-fonts-are-still-missing.454493/). QAチームが反応しているので, 時間がかかるかもしれませんがバニラにおける問題もそのうち修正されるかもしれません.
    -   特にパークスキルの説明文に関しては英文の時点でおかしいものが多いので先にそっちを指摘してあげるべきかもしれません
-   バージョン番号は飛ぶことがあります. 致命的なミスを見つけて非公開にする必要があるとか, こちら側でのバージョン管理の都合などが理由です

# How to Install (&gt; v0.9.1)

Currently M&B localization system is quite limited.

1.  Download one of the Correct Localization JP-\* except “Correct Localization JP - Text” in the Main Files
2.  Unzip it and extract two `CorrectLoacalizationJP-*` folders, drag and drop them into the Modules folder.
3.  Activate the both mods on the launcher and start to play.

-   Each variant is equivalent except the Japanese font
-   Note that modding disables to unlock the achievements. I recommend using [AchievementUn(bloc)ker](https://www.nexusmods.com/mountandblade2bannerlord/mods/4587)
-   Otherwise you can overwrite the vanilla modules by overwriter versions in the Optional Files.

## How to update

It’s easier to donwload “Correct Localization JP - Text” and overwrite the older one.
